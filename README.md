# Food Picker

## Overview

The Food Picker is a simple web application built using HTML, CSS, JavaScript, and React. It allows users to input their restaurant choices, including fast food options. The application provides the functionality to randomly select a choice from the given options using an interactive spinning wheel.

## Features

- **Add Choices:** Users can input and add their restaurant choices to the list.
- **Random Selection:** Once choices are added, the application can randomly select a restaurant for the user.
- **Interactive Wheel:** A visually appealing wheel displays the choices, and users can click "spin" to initiate the random selection.

## Technologies Used

- HTML
- CSS
- JavaScript
- React
